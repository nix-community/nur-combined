## Shelvacu's copyparty server

- This is running [copyparty](https://github.com/9001/copyparty)
- Most things are read-only, except
  - [/ppl/{your username}](/ppl) is just for you. Put whatever you want here.
  - Everyone can write to [/chaosbox](/chaosbox). Expect chaos.
- Click on the file size to select files, then look in the bottom right for "share", which allows you to show files with anyone that has the link.
- Check out [the video intro](/showcase-hq.webm)

## Neat tricks

- Listen to [an album](/chaosbox/2%20Mello%20-%20bop%20it!%20The%20Video%20Game%20(The%20Original%20Soundtrack)/#af-4dc4d73f) on your smartphone and enjoy album-art + controls on the lockscreen
  - also works on desktops, including keyboards with play/pause keys
- for old browsers: [simple](?b=u) or [plain](?ls=t) directory listings
- type `?` to see hotkeys
