# Security Policy

## Supported Versions

Currently, only the latest release of Instant Eyedropper Reborn (IE-R) is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of IE-R seriously. If you believe you have found a security vulnerability, please do not open a public issue. Instead, report it directly to the author.

**Contact:**  
Konstantin Yagola  
Email: [webmaster@spicebrains.com](mailto:webmaster@spicebrains.com)

Please provide a detailed description of the vulnerability and steps to reproduce it. We will acknowledge your report within 48 hours and work on a fix as soon as possible.

Thank you for helping keep IE-R secure! 🦾🛡️
