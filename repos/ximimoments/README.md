# nur-packages

My personal [NUR](https://github.com/nix-community/NUR) repository.

## Packages

This repository contains my custom Nix packages:

- **katifetch**: A lightweight system fetch script.

## Usage

To use these packages, add this repository to your `flake.nix` or `configuration.nix` following the instructions on the [NUR website](https://nur.nix-community.org/).
