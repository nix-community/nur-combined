# Godot MCP Reference

| Tool | Action | Requirements |
| :--- | :--- | :--- |
| `create_scene` | Creates `.tscn` | `scenePath`, `rootNodeType` |
| `add_node` | Adds Node | `scenePath`, `nodeType`, `parentNodePath` |
| `save_scene` | Writes changes | `scenePath` |
| `run_project` | Executes game | `projectPath` |
| `launch_editor` | Opens editor | `projectPath` |
| `get_project_info` | Reads metadata | `projectPath` |
