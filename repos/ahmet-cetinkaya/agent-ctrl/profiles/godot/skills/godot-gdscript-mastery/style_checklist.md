# GDScript Style Checklist

- [ ] `extends` is first line
- [ ] Static typing used everywhere (`: int`, `-> void`)
- [ ] Signals are typed
- [ ] Node paths use `@onready` and `%UniqueNames`
- [ ] Function names are `snake_case`
- [ ] Private methods/vars prefixed with `_`
- [ ] No `get_node()` inside `_process()` or loops
